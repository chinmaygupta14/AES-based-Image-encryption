ABOUT THE AES ALGORITHM
This is an image encryption algorithm based on the combination of the chaos sequence and the modified AES algorithm. In this method, the encryption key is generated by a chaos sequence. Then, the original image is encrypted using the modified AES algorithm and by implementing the round keys produced by the chaos system. It is based on the paper as mentioned in [1].

This approach not only reduces the time complexity of the algorithm but also adds the diffusion ability to the algorithm, which makes the encrypted images resistant to differential attacks. This way of encryption is also resistant to brute force attacks, statistical attacks and entropy attacks.

It is more secure and faster when compared to the normal AES encryption because the introduction of a chaotic system to generate keys makes the algorithm very sensitive to changes ( in the input or the initial values), ie, a minimal change in these values causes a huge difference in the output(encrypted image). The operation block will be of the same size as a normal AES system but a few changes will be introduced in the steps in between.
